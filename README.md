# troposim

Simulate tropospheric noise for InSAR data


## Description

Contains functions to simulate tropospheric turbulence, stratified tropospheric noise.

## Usage

```python
from troposim import turbulence
noise = turbulence.simulate(shape=(500, 500))

```
